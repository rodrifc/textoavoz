Convertir texto a voz MP3

Herramienta web mantenida por la Prof. Laura Cañas y el Prof. Rodrigo Cañas. Este sintetizador utiliza tecnología de texto a voz desarrollada por Microsoft. Su uso es gratuito y el código fuente del sitio se encuentra disponible aquí.

⚠️ ATENCIÓN: Se requiere navegador Edge (Clic para más información)

Debido a limitaciones impuestas por Microsoft esta herramienta únicamente funciona con el navegador web Edge. El proceso de conversión fallará si se utiliza otro programa como Chrome o Firefox. En los sistemas Windows 10 y 11 el navegador Edge ya está instalado. Si usás Mac o Linux podés descargarlo haciendo clic aqui.

Instrucciones (Clic aquí para más información)

Este programa convierte texto escrito en audio MP3. Siga los pasos en pantalla para elegir el texto y la voz que desee.

Puede ingresar texto manualmente o subir un archivo. El sistema acepta archivos TXT, PDF, Word (.docx), FB2 y EPUB.

Tenga en cuenta que si el documento original (como un PDF) tiene imágenes o formatos complejos, es recomendable revisar que el texto se haya extraído de corrido para que la lectura sea natural.

Cargar Pronunciaciones (.lexx): Si el motor pronuncia mal alguna sigla o palabra técnica, usted puede cargar un archivo .LEXX con correcciones. Se crea con el Bloc de notas. Por ejemplo:

  "COPILOT"="ko-pai-lot"
  "A.D.N"="a-de-ene"
